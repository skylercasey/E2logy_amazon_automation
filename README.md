# E2logy_amazon_automation




This is the python Pytest framework with selenium  for automating cart in amazon.com

1.For performing this first you have to install python in your PC
   Visit the official Python website at https://www.python.org/downloads/. You'll see the latest versions available for download. Choose the version you want to install. For most users, it's recommended to download the latest stable version.

2.Verify Installation:
After the installation is complete, you can verify it by opening a command prompt (on Windows, press Win + R, type cmd, and press Enter) and typing python --version or python3 --version. This command should display the installed Python version.

3. Open the cmd
4. Go to the desired folder
5. create a python environment using the command " python -m venv amazon_env"
  (here amazon_env is the name of environment)
6. A directory will be created with amazon_env

7. execute this command to activate environment "amazon_env\scripts\activate" and enter
8. again execute this command "pip install selenium"
9. install framework using this command " pip install pytest"
10. Run this code using this command " pytest -rA"
